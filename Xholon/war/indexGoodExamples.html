Xholon - Good JS workbook examples

This is a curated list of the best examples to look at when learning to create your own Xholon JavaScript-based workbooks. All example workbooks are available as gist text files (xholonWorkbook.xml) at gist.github.com/kenwebb/ . Each example in the following list includes a link to the Xholon editor for that workbook (Preview,Edit), and a link to the Xholon runtime (Run) for that workbook. The workbooks are listed from newest at the top, to oldest at the bottom of the page. In many cases, you will need to read the preview (the notes and source code) to learn what to expect from an app, before running it. Many of the examples write text to the "out" tab. In some of the examples, you should click the "Step" button (or press the "S" key), one or more times, to get things to happen.

You can open or view workbooks that have been saved as public gists in GitHub. If you have a GitHub account, you will also be able to save a new or edited workbook as a gist. To open or view a gist, you will need to know the gist id, which you can find by searching GitHub for XholonWorkbook. If you don't know what GitHub is, then just ignore this part.
