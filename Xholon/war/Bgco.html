  1. Run the application (app).
  2. Select the clipboard tab on the app page. Note the various control icons at the bottom of the page.
  3. Copy all the data from one of the asana data files (you must be logged into asana to do this).
  4. Paste the data into the app's clipboard tab (this will take a few seconds to complete).
  5. Right-click the light-green square ("Drop new Members here") and select Edit > Paste Last Child .
  6. Click the app's Start button.
  7. Watch.
  8. You can Pause and Unpause the app by clicking the Pause icon.
  9. You can Pause and Unpause just the animation by clicking the Toggle Animation icon.

Or you can run App 2 with the Leaflet library enabled, to display clubhouses and activity as an overlay on a map of Ottawa.
