Furcifer
What is it

The Furcifer application is used for QUnit unit testing of the Xholon JavaScript API. The word "Furcifer" refers to a genus of chameleons, and especially to the panther chameleon (Furcifer pardalis) which is probably the largest species of chameleon. In the context of the Xholon project, the name signifies that this app is an expanded (larger) version of the Chameleon app.

How to use it

To start the test cases using Firebug or Developer Tools: xh.test();

To start the test cases from within the Xholon app: <script>$wnd.xh.test();</script>
