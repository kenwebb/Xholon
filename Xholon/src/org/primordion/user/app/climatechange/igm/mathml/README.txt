ASCIIMathML
 - ASCII to MathML presentation
 - can use Latex or simple ASCII
 - example:
   Find any PNG formula in wikipedia,
   select "View image info" in Firefox,
   copy "Associated text", (it's in Latex syntax)
   paste into ASCIIMathML Editor
 - http://www1.chapman.edu/~jipsen/mathml/asciimath.html

SVGMath python library
 - MathML presentation to SVG
 - http://grigoriev.ru/svgmath/
 - this works:
python ~/SVGMath-0.3.3/math2svg.py -s ~/workspace/Xholon/src/org/primordion/user/app/climatechange/igm/mathml/formula1.mml -o out.svg
 - can paste the result into a SVG file, and then edit the file using Inkscape

Open Office has a formula editor
 - I used it to produce formula1.mml and .odf
