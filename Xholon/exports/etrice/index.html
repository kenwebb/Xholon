Xholon to eTrice

Xholon models can export themselves to eTrice format, using the initial and very-incomplete version of the Xholon2Etrice exporter. eTrice is an independent implementation of the same Real-Time Object-Oriented Modeling (ROOM) approach that Xholon is ultimately derived from. The Xholon exporter generates .room files that can be read by the eTrice tool. This page includes links to a number of generated (and lightly-edited) .room files.
